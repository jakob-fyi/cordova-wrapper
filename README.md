# Cordova Wrapper Sample App

## Generate Resources (Icon & Splash Screen) 

Package: https://github.com/ionic-team/cordova-res

Install: `sudo npm install -g --unsafe-perm cordova-res`

```sh
cordova-res android
cordova-res ios
```
